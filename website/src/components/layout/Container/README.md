# Container

## Purpose

Provides a centered layout wrapper for all pages.

## Features

- Responsive width
- Centered content
- Consistent horizontal padding

## Props

| Prop | Type | Description |
|------|------|-------------|
| children | ReactNode | Component content |
| className | string | Additional Tailwind classes |

## Used By

- Home
- Lessons
- Labs
- Projects
- About

## Future Improvements

- Container sizes (sm, md, lg)
- Fluid layout
- Vertical padding options
