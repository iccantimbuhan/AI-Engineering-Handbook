# Surface

Version: 2.0

---

## Purpose

Surface is the foundational layout container used throughout the AI Engineering Platform.

Unlike Card, Surface is not interactive.

It exists to group related content while maintaining visual rhythm and generous whitespace.

---

## Design Principles

- Editorial layout
- Calm appearance
- Large spacing
- Minimal elevation
- No hover interaction

---

## Visual Rules

- Radius: 20px
- Padding: 32px
- Background: White
- Border: Soft Slate
- Shadow: Extra subtle

---

## Used By

- Documentation Sections
- Dashboard Sections
- Lesson Content
- Callouts
- Empty States

---

## Never Use For

- Clickable cards
- Navigation items
- Interactive widgets

Use Card instead.
