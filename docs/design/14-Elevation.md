# Elevation

Version: 1.0

---

# Philosophy

Depth should communicate hierarchy.

Elevation should never become decoration.

The platform should feel lightweight.

# Borders

Cards should rely on subtle borders before shadows.

Borders should remain soft.

---

# Hover

Hover should slightly increase elevation.

Movement should remain subtle.

---

# Design Rules

- Prefer borders over heavy shadows.
- Avoid dramatic depth.
- Elevation should communicate interaction.
