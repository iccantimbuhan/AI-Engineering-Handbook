# Playground

Version: 1.0

# Purpose

The Playground is an experimentation environment.

Learners can safely test prompts, APIs, models, and components without affecting lessons.

Experiment with prompts.

## Model Playground

## API Playground

## Component Playground

Preview reusable UI components.
