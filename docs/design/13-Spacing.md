# Spacing

Version: 1.0

---

# Philosophy

Whitespace is a feature.

Spacing creates hierarchy, improves readability, and reduces cognitive load.

The platform should never feel crowded.

---

# Base Unit

The design system uses an 8-point spacing scale.

```
4
8
12
16
24
32
40
48
64
80
96
```

Every component should follow this scale.

---

# Section Spacing

Between major page sections

96px

Between content blocks

64px

Between headings and content

32px

Between paragraphs

24px

Between related controls

16px

---

# Card Spacing

Internal padding

24px

Large cards

32px

Compact cards

20px

---

# Layout Spacing

Sidebar

24px

Header

24px

Main Content

48px

Page Margin

48px

---

# Design Rules

- Never use arbitrary spacing.
- Follow the spacing scale.
- Larger spacing improves readability.
- Consistency is more important than precision.
