# Motion

Version: 1.0

---

# Philosophy

Motion communicates change.

Animation should never exist purely for decoration.

Users should rarely notice animations consciously.

---

# Duration

Fast

150ms

Standard

200ms

Slow

300ms

Avoid animations longer than 400ms.

---

# Easing

Use smooth easing for all transitions.

Avoid bounce animations.

---

# Hover

Hover effects should be subtle.

Examples:

- Background fade
- Slight elevation
- Border emphasis

---

# Expand / Collapse

Expandable content should animate height and icon rotation.

Animations should feel responsive.

---

# Navigation

Page transitions should remain lightweight.

Avoid excessive movement.

---

# Loading

Use skeleton loaders whenever practical.

Avoid unnecessary loading spinners.

---

# Design Rules

Animation should improve usability.

Never animate large sections unnecessarily.

Respect reduced motion preferences whenever possible.
