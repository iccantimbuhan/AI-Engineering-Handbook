# Handbook

Version: 1.0

---

# Purpose

The Handbook is the core learning experience of the AI Engineering Platform.

It provides structured lessons that guide learners from beginner to production-level AI Engineering.

---

# Content Source

All lessons are generated from Markdown.

```
docs/

↓

Generator

↓

content.json

↓

React Application
```

No lesson should be manually added to React.

---

# Lesson Structure

Every lesson consists of:

- Breadcrumb
- Lesson Title
- Description
- Metadata
- Reading Time
- Markdown Content
- Previous Lesson
- Next Lesson

---

# Sidebar

The sidebar automatically displays all modules and lessons.

Navigation is generated from Markdown metadata.

---

# Reading Experience

Every lesson should support:

- Table of Contents
- Reading Progress
- Copy Code
- Syntax Highlighting
- Responsive Images
- Mermaid Diagrams (future)
