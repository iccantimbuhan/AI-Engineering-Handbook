# Buttons

Version: 1.0

---

# Philosophy

Buttons should communicate actions clearly.

The platform should avoid unnecessary button styles.

---

# Variants

Primary

Main action.

Secondary

Supporting action.

Ghost

Low emphasis.

Link

Navigation.

Danger

Destructive actions.

---

# Border Radius

14px

---

# Height

Small

36px

Medium

44px

Large

52px

---

# Icon Spacing

12px

---

# States

Default

Hover

Focus

Active

Disabled

Loading

---

# Interaction

Hover

Slight background change.

Active

Subtle scale.

Focus

Visible focus ring.

Disabled

Reduced opacity.

Loading

Spinner replaces icon.

---

# Design Rules

- One primary button per section.
- Avoid multiple competing actions.
- Buttons should remain readable.
